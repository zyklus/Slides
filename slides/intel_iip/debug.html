Other Debuggers
Both the Chrome and Internet Explorer debuggers have similar functionality to Firebug.

Google Chrome

Comes with a debugger built in. To access it, go to View --> Developer --> Developer tools or press:
  • Windows: ctrl + shift + i
  • Mac: ⌥ + ⌘ + i

Internet Explorer

IE8 and IE9 both come with build in debuggers. To access, go to View --> Developer Tools or press:
  • F12
Note: The Internet Explorer is buggy. It is strongly suggested that most debugging be done in either Firefox or Chrome, and the Internet Explorer debugging tools used to solve IE-specific issues.

One of the main issues is that IE does not display updated HTML since the debugger was open. This means popup content, etc will not be displayed unless they were opened before the debugger was opened.

Console Debugging

Echo basic messages

console.log( any_object );

Show a stack trace

console.trace()

Count calls to a specific event

console.count( event_title );

Group console statements

console.group();
console.groupEnd();
Note: Do not leave console statements in production code as it will cause JavaScript errors in browsers without debuggers enabled
