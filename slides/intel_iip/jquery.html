The $ Function
jQuery's $ function is overloaded in several ways:

As a selector

var elements = $('#myID span');

As a DOM Creation Utility

var node = $('<div id="myID"><span>Hello, World!<span></div>');

As an onload handler

$(function(){
  // Any code put here will fire when the document is loaded
});
CSS 2/3 Selectors
In addition to fully supporting all of the CSS1 selectors, jQuery fully supports all CSS2/3 selectors regardless of whether or not the current browser supports them. These selectors can not be used in stylesheets due to the fact that IE7 does not support them.

Immediate Child Selector

html: <div><span><span></span></span></div>
css : div > span { ... }

Next Sibling Selector

html: <div></div><span></span><span></span>
css : div + span { ... }

Attribute Selector

html: <input type="radio" />
css : input[type="radio"]

:checked Pseudo-Selector

html: <input type="checkbox" checked />
css : input:checked { ... }
Reference: CSS Selectors
More CSS 2/3 Selectors

:enabled and :disabled Selectors

html: <input disabled />
css : input:disabled { ... }

:not Selector

html: <input disabled />
css : input:not(:enabled) { ... }

:even and :odd Selectors

html: <ul>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
  </ul>

css : ul li:odd { ... }            /* jQuery-specific syntax */
css : ul li:nth-child(odd) { ... } /* normal CSS syntax */
